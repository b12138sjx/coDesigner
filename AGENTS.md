# Agent 指引（CoDesigner）

面向在 Cursor 中参与本仓库开发的 AI Agent。

- **代码库约定**：使用项目 Skill `coDesigner-codebase`（`.cursor/skills/coDesigner-codebase/`），遵循其中技术栈、目录与画布/文档/接口说明。
- **代码风格**：遵循 `.cursor/rules/project-conventions.mdc`（React、Zustand、CSS Modules、`@/` 路径）。
- **设计稿与浏览器**：用户提供 Figma 链接或要求在浏览器中验证时，遵循 `.cursor/rules/design-and-mcp.mdc`，使用 Figma MCP 或 Chrome DevTools MCP。
- **更多说明**：见 [docs/README.md](docs/README.md)、[docs/TOOLS_AND_MCP.md](docs/TOOLS_AND_MCP.md)。
